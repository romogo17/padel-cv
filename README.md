# padel-cv

This project explores two approaches for classifying padel strokes from video:

1. **Pose Estimation + Classification** - Extract player pose keypoints, then classify strokes from skeletal data
2. **Direct Video Classification** - End-to-end classification directly from video frames

The dataset combines:

- **Professional matches** - YouTube videos from Premier Padel tournaments
- **Amateur recordings** - Amateur matches from Costa Rica
